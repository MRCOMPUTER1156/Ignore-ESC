# ESC Ignore Mod

Prevents the ESC key from functioning in-game, helping to avoid accidental pauses.

## Features

* Completely disables the ESC key
* Allows uninterrupted gameplay
* Works in real time

## Controls

* F1 → Toggle the mod on/off

## Notes

* If you are unable to pause the game, make sure to disable the mod using F1
* May be inconvenient if you need to exit a level quickly
